Tk Widget Commands that are supported in Swank

These commands create visible widgets on the screen and are designed to work very similarly to the corresponding Tk widgets. They may not, however, have exactly the same behaviour or configuration options as the Tk widgets. The links are to the original Tk documentation. After each command name, is a link to the Java Swing widget that is used for the acutal implementation in Swank.

canvas JPanel message JTextArea tk_dialog JOptionPane
checkbutton JCheckBox panedwindow JSplitPane tk_getOpenFile JFileChooser
entry JTextField radiobutton JRadioBox tk_getSaveFile JFileChooser
frame JPanel scale JSlider tk_optionMenu JComboBox
label JTextField scrollbar JScrollbar tkerror
labelframe JPanel spinbox JSpinner tk_messageBox JOptionPane
listbox JList text JTextPane toplevel JFrame

Tk Commands that are present in Swank but do nothing

These commands are basically stubs. They don't do anything, but their presence helps avoid some errors that would occur in running Tk scripts if they didn't exist.
option grab

Swank Widget Commands that are not present in Tk

These commands either don't exist in Tk, or their behaviour is fairly different from a similar Tk command. Some, like jfilechooser, are used to implement Tk commands like tk_getOpenFile.
